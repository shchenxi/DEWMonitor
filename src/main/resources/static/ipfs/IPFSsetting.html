不会收集任何 CID 、文件名或个人信息。我们仅收集度量数据来帮助我们优化后续工作的优先级，同时收集系统配置信息引导我们进行测试。

配置收集哪些数据

Protocol Labs 运行了 Countly 实例帮助我们收集这个应用产生的匿名数据。

你可以选择发送以下数据：

– 何时使用了此应用，使用了多长时间，以及相关的浏览器信息。
– 添加或删除文件等操作以及花费的时间
– 使用应用的哪些页面
– IP 地址的国家代码（ IP 地址将会被丢弃）
– JavaScript 错误消息和堆栈跟踪
